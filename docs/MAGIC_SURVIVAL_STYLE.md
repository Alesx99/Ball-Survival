# Progressione stage nel tempo (modalità Standard)

In **modalità Standard** la partita funziona così:

- Partenza sempre da **Stage 1** (Pianura Eterna). La selezione stage nel menu non è disponibile.
- **Progressione stage basata sul tempo**: ogni 4 minuti di gioco si avanza allo stage successivo (2 → 3 → … → 8). Non servono obiettivi di crafting o kill per avanzare.
- La card "Stage" nel menu è nascosta in Standard; riappare per Endless, Daily, Boss Rush, Tutorial.

Parametro in `StageSystem.js`: `MINUTES_PER_STAGE = 4`.
